# SAMI — PRD v1

## 1. Product Vision
SAMI — дисциплинарное social fitness приложение для людей 25–45, которые хотят вернуть структуру в тело и день.

Принципы:
- Calm UX, без дофаминового шума
- Ритм и последовательность > мотивационные всплески
- Простые ритуалы, которые реально делать ежедневно

## 2. Goals (MVP)
- Запустить iOS MVP в App Store
- Получить первые активные пользователи и данные retention
- Проверить гипотезу: timeline + structured creator + social feed повышают вовлечённость

## 3. Primary Users
- Взрослые 25–45
- Не про «фитнес-инфлюенсерство», а про повседневную дисциплину
- Интересы: bodyweight, mobility, breathwork, running, boxing basics

## 4. Core Features (MVP)
1. **Home (Timeline + Calendar)**
   - Прошлые/будущие активности
   - Today plan
   - Completion state
2. **Community Feed**
   - Лента тренировок
   - Фильтры: тип, длительность, мышечная группа, сложность
3. **Workout Creator (Structured)**
   - Title/category/difficulty/duration
   - Exercises list (reorder)
   - Sets/reps/rest/media
   - Publish (public/private)
4. **Profile**
   - Личные метрики и streak
   - Сетка опубликованных тренировок
   - Preferences

## 5. Non-Goals (v1)
- Live workouts
- Сложная AI-персонализация
- Полный social graph (follows/comments/reactions at scale)

## 6. UX Direction
- Minimal, Apple-like, neutral palette
- Large readable typography
- Rounded cards, soft depth
- Микроанимации только функциональные

## 7. Success Metrics
- D1 / D7 retention
- Weekly completion rate
- % users who create at least 1 workout
- Feed engagement (open rate / save rate)

## 8. Risks
- Перегрузить MVP social-фичами
- Слабая структурность creator-flow
- Отсутствие чёткой event taxonomy с первого дня
