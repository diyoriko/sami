# SAMI — MVP Scope

## In Scope (v1)
- Auth (email / magic link)
- Onboarding (goal, level, focus, available time)
- Timeline home + compact calendar strip
- Daily activity cards with completion state
- Feed with filters
- Workout creator wizard (4-step)
- Profile with core stats
- Push reminders (basic)
- Analytics baseline events

## Out of Scope (v1)
- Live sessions
- Full comments/threading
- AI workout generation
- Deep HealthKit sync
- Complex monetization experiments

## Release slices
### Slice A (Week 1)
- Auth + onboarding + home shell

### Slice B (Week 2)
- Timeline logic + completion

### Slice C (Week 3)
- Feed + filters

### Slice D (Week 4)
- Workout creator + publish + profile + QA

## MVP Quality Bar
- App must feel calm, clean, reliable
- No broken navigation
- Core flows < 3 taps from home
- Empty states designed (not blank)
