## SPRINT 2 — Библиотека и взаимодействие (неделя 15-21 марта)

### P1: Фильтрация видео через бота
Пользователь пишет боту и может найти нужную тренировку.

- [ ] Команда `/find` или inline-кнопки: выбор категории → длительность → сложность
- [ ] Фильтры: категория (стретчинг/силовая/мобильность), длительность (5-10 / 10-20 / 20+ мин), сложность (начинающий/средний/продвинутый), группа мышц, наличие инвентаря
- [ ] Результат: бот отправляет ссылку на видео в канале или forward
- [ ] Индексация: все опубликованные видео сохраняются в БД с метаданными
- [ ] Поиск по тексту: `/search мобильность бёдра 10 минут`

### P1: Рубрики и контент-календарь
- [ ] `#ритуал_недели` (пн) — одна практика-фокус на всю неделю
- [ ] `#механика` (ср) — разбор упражнения: техника, ошибки, зачем

- [ ] Образовательный текстовый пост 1x/неделю (без видео)

### P1: Первый челлендж "7 дней мобильности"

## SPRINT 3 — UGC и community (неделя 22-28 марта)

### P1: Пользовательский контент (UGC)
Участники постят свои видео через бота — единый стиль оформления.

- [ ] Команда `/submit` — участник отправляет видео боту
- [ ] Бот запрашивает метаданные: категория, сложность, мышцы, инвентарь
- [ ] Бот форматирует в едином стиле канала (заголовок, теги, структура)
- [ ] Модерация: видео попадает в очередь → admin одобряет → публикация
- [ ] Автор получает упоминание в посте
- [ ] UGC-видео помечаются тегом `#от_участников`

### P2: Обсуждение видео
- [ ] Привязать группу к каналу как комментарии (Telegram Discussion Group)
- [ ] Бот приглашает к обсуждению после каждого видео
- [ ] Вопрос-триггер под видео: "Делал? Как ощущения?"

### P2: Buddy invite
- [ ] После 3-го чекина бот предлагает: "Знаешь кого-то, кому это нужно?"
- [ ] Персональная invite-ссылка с трекингом
- [ ] Признание за приглашённых друзей

## Качество — приоритет

Каждый элемент (тексты, механики, UX бота) должен соответствовать бренду:
- Тон: спокойно, конкретно, методично
- Без: "ПОГНАЛИ", "жиросжигание", "до/после"
- Стиль: тренер-методист, не инфлюенсер
- One-liner: "Не мотивация. Структура."
