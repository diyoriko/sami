## P1 — Активация новых участников

После капчи участник "висит" без действия. Нужно вести к первой тренировке.

- [ ] Welcome-сообщение после капчи: приветствие + "посмотри сегодняшнюю тренировку"
- [ ] Goal quiz: цель (стретч / сила / мобильность) → уровень → персональная рекомендация
- [ ] Автоответ на результат quiz: "Сегодня в 12:00 силовая — идеально для тебя"
- [ ] Закреплённое сообщение в группе: кто мы, что здесь происходит, как участвовать

---

## P1 — Чекин как двусторонний диалог

Текущий чекин: кнопки "Сделал / Частично / Не получилось" — reply rate низкий.

- [ ] Добавить вопрос дня в чекин ("Что дала сегодняшняя тренировка?")
- [ ] Мягкий ответ бота на "Не получилось" — поддержка, не осуждение
- [ ] Публичное признание: бот отмечает тех, кто 3+ дня подряд
- [ ] Итоги дня: автопост с результатами чекина (сколько сделали, % активных)

## P1 — Рубрики и контент-календарь

Контент однообразный: 3 видео + чекин каждый день. Нужны рубрики для разнообразия.

- [ ] `#ритуал_недели` (пн) — одна практика-фокус на всю неделю
- [ ] `#механика` (ср) — разбор упражнения: техника, ошибки, зачем
- [ ] `#прогресс_пятницы` (пт) — итоги недели, публичное признание активных
- [ ] `#за_кулисами` (вс) — что происходит с проектом, прозрачность
- [ ] Недельный digest по воскресеньям — итоги + план на следующую неделю
