# Sami Community Agent

Telegram-бот для ежедневной публикации тренировочных видео в канал Sami.

## Что делает

- **19:00** — ищет YouTube-видео (3 категории × 3 варианта), отправляет тебе в личку на апрув
- **08:00** — постит одобренное видео по стретчингу
- **12:00** — постит силовую тренировку
- **17:00** — постит мобильность
- **21:00** — постит вечерний чекин с кнопками ✅ / 😅 / ❌
- **23:55** — пишет дневной отчёт для агента-стратега

## Модерация группы

- Новые участники → приветствие + опрос о цели (фильтр от ботов)
- Новые участники первые 24ч — нельзя постить ссылки
- Спам → предупреждение → мут 24ч → бан
- Команда `/report` для репорта спама

## Установка

```bash
cd agents/community
npm install
cp .env.example .env
# Заполни .env своими токенами
npm run dev          # разработка
npm run build && npm start  # продакшн
```

## Что нужно подготовить

1. **Telegram**: создать канал `@sami_daily` + группу `@sami_chat` → привязать → добавить бота как админа
2. **Bot Token**: получить от `@BotFather`
3. **YouTube API Key**: Google Cloud → YouTube Data API v3 → Create API Key
4. **Твой User ID**: написать `@userinfobot`

## Команды (только для администратора)

| Команда | Действие |
|---------|----------|
| `/status` | Статистика чекинов за сегодня |
| `/search` | Запустить поиск видео вручную |
| `/post` | Опубликовать все посты сейчас |
| `/post stretching` | Опубликовать только стретчинг |
| `/checkin` | Опубликовать чекин вручную |

## Синхронизация со стратегом

- Читает `reports/strategist/.internal/latest.json` → берёт ключевые слова для поиска
- Пишет `reports/community/.internal/latest.json` → стратег читает на следующем цикле
