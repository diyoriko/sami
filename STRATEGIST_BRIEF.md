# STRATEGIST BRIEF

Цель: построить Telegram-сообщество вокруг Sami так, чтобы сообщество стало каналом дистрибуции и валидатором гипотез до запуска приложения.

Что важно:
- Сообщество должно генерировать инсайты для roadmap приложения.
- Сообщество должно накапливать waitlist ранних пользователей.
- Коммуникация должна быть полезной сама по себе, не только промо.

Текущие ограничения:
- Приложение в стадии подготовки MVP (FlutterFlow + Supabase).
- Community agent работает на Railway, постит 3 видео/день + вечерний чекин.

## Бренд SAMI — идентичность (из Figma)

### Миссия
> «Даём тёплую, честную и эстетичную среду для заботы о теле. Нужен только коврик.
> Двигайся красиво и просто — для себя, вместе с нами.»

Механика лояльности: ты начинаешь ради здоровья → возвращаешься ради привычки → делишься, потому что тебе нравится, как это выглядит и чувствуется.

### Ценности (дословно из Figma)

| Ценность | Суть | Как проявляется |
|---|---|---|
| Жить лучше, а не быстрее похудеть | Цель — качество жизни, не вес | Без контента про похудение, жиросжигание, «до/после» |
| Тело = партнёр, не проект | Позитивная лексика | Без «исправь недостаток», без «идеального тела» |
| Красиво во всём | Эстетика движения и UI | Светлая палитра, честные фото, минимализм |
| Поддержка, не соревнование | Совместность без рейтингов | Эмодзи-реакции, челленджи без таблиц лидеров |
| Нужен только коврик | Доступность | Старт с одного тапа, только bodyweight |
| Реальные люди и результаты | Антиглянец | Пот, усилие — без ретуши и позирования |

### Архетипы и Tone of Voice

| Слой | Архетип | Пример голоса |
|---|---|---|
| Поддержка | Опекун | «6 минут, чтобы плечи сказали спасибо» |
| Честность | Мудрец | «Пульс поднимется на ~20 уд/мин — нормально, держи темп» |
| Красота | Любовник | «Движение выглядит так же хорошо, как ты себя чувствуешь» |
| Воодушевление | Герой | «Катану прогресса точим? Давай ещё один выпад» |

**Голос:** спокойно, конкретно. Не агрессивный мотивационный сленг. Тренер-методист, не инфлюенсер.

### Позиционирование vs конкуренты

| Параметр | SAMI | Типовой фитнес-app |
|---|---|---|
| Оборудование | Только коврик | Требуется инвентарь |
| Старт-барьер | One-tap, где ты | 5-экранный онбординг, поездка в зал |
| Визуальный тон | Zen-minimal | Неон/градиенты |
| Соц-слой | Поддержка, UGC без фильтров | Рейтинги, «идеальные» тела |
| Голос | Спокойно, конкретно | Агрессивный мотивационный сленг |

### Манифест (суть)

- Дисциплина — не оковы, а пульс
- Красота рождается из дисциплины, а не из идеальной формы
- Пот и ошибки — доказательство движения, а не дефект
- Поддержка звучит тише аплодисментов, но держит темп лучше
- «Потом» уже вплетено в «сейчас» — фиксируем мгновения
- Город — зал без стен: лестница, площадь, крыша

### Визуальная идентичность

**Фотография:**
- Настоящие люди в настоящем движении
- Пот, усталость, усилие — без ретуши и позирования
- Динамичные кадры: напряжённые мышцы, резкость в движении
- Без глянца. Меньше фильтров, больше правды

**Дизайн:**
- Минимализм. Белый фон, простые формы, чистое пространство
- Акценты в движении: линии, энергия, свет
- Одно ощущение: ты в процессе, ты в теле, ты в фокусе

**Типографика:**
- Primary: PP Right Grotesk Wide — структура и душа, нейтральность + индивидуальность
- UI: SF Pro — чистота, тепло, читаемость (стандарт Apple)
- Mono: SF Mono / Cera Mono — для данных и технических элементов

**Логотип:** «сами» / «SAMI» / «SAM/I» — финальная версия на основе итераций for.me → Luchee → Сами

**Маркетинг:**
- Инфлюенсеры: микро-блогеры и тренеры с честным контентом (не глянец)
- Соц.сети: тренировки без фильтров, реальные эмоции
- Community — главный инструмент распространения и удержания

## Статус сообщества

Community Agent активен на Railway. Канал `@sami_daily` + группа `Sami Community`.
Расписание: все 3 видео одновременно в 08:00 (стретчинг + силовая + мобильность) + чекин 22:00.
Видео скачиваются через yt-dlp и заливаются напрямую в Telegram (работает без VPN в РФ).
Поиск видео: приоритет только коврик, 1 кандидат на категорию, бренд-скоринг по ценностям Figma.
Метрики: analytics отчёт доступен через `reports/analytics/.internal/latest.json`.

## Приоритеты проекта (текущие)

**ГЛАВНЫЙ ПРИОРИТЕТ — рост и вовлечённость сообщества.**
Мобильное приложение и лендинг — не в приоритете до тех пор, пока сообщество не станет
достаточно активным (целевой порог: стабильный daily active engagement, waitlist > 500).

Фокус стратега:
1. Активация новых участников (captcha → goal quiz → первая тренировка)
2. Retention через ежедневные чекины и контент
3. Рост waitlist через organic sharing
4. Сбор инсайтов для будущего roadmap приложения

## Community Packet (ОБЯЗАТЕЛЬНО производи в каждом отчёте)

В конце каждого отчёта добавляй машиночитаемый блок:

```json
// COMMUNITY_PACKET_START
{
  "week_focus": "<одна из: стретчинг|силовая|мобильность|общая>",
  "content_themes": ["тема1", "тема2", "тема3"],
  "challenge_active": false,
  "challenge_name": null,
  "search_keywords": {
    "stretching": ["ключевое слово для поиска на YouTube"],
    "strength": ["ключевое слово для поиска на YouTube"],
    "mobility": ["ключевое слово для поиска на YouTube"]
  },
  "community_priority": "<одна из: activation|retention|waitlist|feedback>"
}
// COMMUNITY_PACKET_END
```

Ключевые слова должны быть на русском языке и отражать ценности бренда:
только bodyweight/без инвентаря, спокойный тон, без похудения и хайпа.
