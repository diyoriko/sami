# STRATEGIST BRIEF

Цель: построить Telegram-сообщество вокруг Sami так, чтобы сообщество стало каналом дистрибуции и валидатором гипотез до запуска приложения.

Что важно:
- Сообщество должно генерировать инсайты для roadmap приложения.
- Сообщество должно накапливать waitlist ранних пользователей.
- Коммуникация должна быть полезной сама по себе, не только промо.

Текущие ограничения:
- Только desktop-прототип/макет продукта.
- Приложение пока в стадии подготовки MVP.

## Статус сообщества

Community Agent активен. Канал + группа в Telegram запущены.
Агент постит 3 видео в день (стретчинг, силовая, мобильность) + вечерний чекин.

## Community Packet (ОБЯЗАТЕЛЬНО производи в каждом отчёте)

В конце каждого отчёта добавляй машиночитаемый блок:

```json
// COMMUNITY_PACKET_START
{
  "week_focus": "<одна из: стретчинг|силовая|мобильность|общая>",
  "content_themes": ["тема1", "тема2", "тема3"],
  "challenge_active": false,
  "challenge_name": null,
  "search_keywords": {
    "stretching": ["ключевое слово для поиска на YouTube"],
    "strength": ["ключевое слово для поиска на YouTube"],
    "mobility": ["ключевое слово для поиска на YouTube"]
  },
  "community_priority": "<одна из: activation|retention|waitlist|feedback>"
}
// COMMUNITY_PACKET_END
```
