# STRATEGIST BRIEF

## Главная идея

Telegram-сообщество SAMI — это **прото-версия будущего приложения**. Не промо-канал, а полноценный продукт: люди заходят, видят библиотеку видео, фильтруют через бота, взаимодействуют, делятся своим контентом. Всё что будет в приложении — сначала тестируется здесь.

## Что должно работать в сообществе (целевое состояние)

### Библиотека видео
- Канал `@sami_daily` = лента тренировок с единым стилем оформления
- Каждое видео: категория, длительность, сложность, группа мышц, инвентарь, хештеги
- Пользователь через бота может **фильтровать** видео: по длительности, мышцам, уровню, наличию инвентаря
- Видео доступны через поиск по тегам в канале

### Пользовательский контент (UGC)
- Участники могут **постить свои видео через бота** (не напрямую в канал)
- Бот проверяет/дополняет теги, форматирует в едином стиле
- Модерация: одобрение перед публикацией
- Это создаёт ощущение "наше сообщество" а не "чей-то канал"

### Взаимодействие
- Обсуждение видео в комментариях (группа привязана к каналу)
- Чекины как ежедневный ритуал: вопрос дня + реакции + признание стриков
- Челленджи, рубрики, buddy invite

### Монетизация (заложить сейчас)
- Продумать модель: premium-контент, ранний доступ к приложению, partnership с тренерами
- Не продавать пока, но заложить фундамент в архитектуру

### Привлечение людей
- Органический рост: UGC → шеринг, челленджи → виральность
- Рубрики как точки входа
- Cross-promotion с микро-тренерами

## Качество — приоритет #1

Всё в сообществе должно быть **очень круто**: механики, UX бота, тексты, оформление. Это не тестовый канал — это продукт. Каждый элемент должен соответствовать бренду.

## Бренд SAMI — идентичность (из Figma)

### Миссия
> «Даём тёплую, честную и эстетичную среду для заботы о теле. Нужен только коврик.
> Двигайся красиво и просто — для себя, вместе с нами.»

### Ценности

| Ценность | Суть | Как проявляется |
|---|---|---|
| Жить лучше, а не быстрее похудеть | Цель — качество жизни, не вес | Без контента про похудение, жиросжигание, «до/после» |
| Тело = партнёр, не проект | Позитивная лексика | Без «исправь недостаток», без «идеального тела» |
| Красиво во всём | Эстетика движения и UI | Светлая палитра, честные фото, минимализм |
| Поддержка, не соревнование | Совместность без рейтингов | Эмодзи-реакции, челленджи без таблиц лидеров |
| Нужен только коврик | Доступность | Только bodyweight |
| Реальные люди и результаты | Антиглянец | Пот, усилие — без ретуши и позирования |

### Tone of Voice

**Голос:** спокойно, конкретно. Не агрессивный мотивационный сленг. Тренер-методист, не инфлюенсер.

| Слой | Архетип | Пример |
|---|---|---|
| Поддержка | Опекун | «6 минут, чтобы плечи сказали спасибо» |
| Честность | Мудрец | «Пульс поднимется на ~20 уд/мин — нормально, держи темп» |
| Красота | Любовник | «Движение выглядит так же хорошо, как ты себя чувствуешь» |

### Позиционирование
- ICP: 28-40, живут в ритме города, хотят регулярности без зала
- Отстройка: bodyweight-first, zen-минимализм, без токсичной мотивации
- One-liner: "Не мотивация. Структура."
